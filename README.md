# D3ModelViewer

**D3ModelViewer**一个基于Three.js的用于在可道云KodExplorer与KodBox中查看3d模型的插件，支持常用的fbx,obj,gltf,glb,bvh,3ds,ply,stl,usdz,3dm文件格式

## 版本说明

* main分支为KodExplorer版本
* kodbox分支为KodBox版本

## 插件安装说明

插件解压后放置在`/plugins`文件夹下，然后登录可道云后台插件中心启用即可。
